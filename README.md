Kaokuro is a simple yet powerful Discord bot focused on utility and some fun commands.

## Features

- Download videos in various formats  
- Control speed of videos or audios (preserving pitch or not)  
- Extract audio from attachments by replying  
- Media player in voice chat (NOT IMPLEMENTED)  
- AI commands (NOT IMPLEMENTED)  

## Quickstart

1. clone this repo  
2. create a `config.yml` (see [configuration docs](docs/config.md))  
3. run the bot with python  

```bash
python main.py
```
