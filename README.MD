Kaoruko is a lovable Discord bot built with discord.py (Python) featuring a modular architecture.

**Features:**
- Download command (in progress)
- Speed command (in progress)

**Installation:**
1. Clone the repository
2. Navigate to the project directory
3. Create a virtual environment
4. Activate the virtual environment
5. Install dependencies and run

```bash
git clone https://github.com/Creeper011/kaoruko
cd kaoruko
make venv
make install
make run
```

**Note:** If you use **pyenv**, run these commands first:
```bash
pyenv install 3.12.5
pyenv local 3.12.5  # in project folder
```
